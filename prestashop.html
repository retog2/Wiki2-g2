¿QUÉ ES?

Es un sistema gestor de contenidos (CMS) libre y de código abierto.

PrestaShop está disponible bajo la licencia Open Software. El software está escrito en PHP y basado en el motor de plantillas Smarty. MySQL es el motor de base de datos predeterminado.

¿PARA QUÉ SIRVE?

Su principal función es la creación de tiendas online de comercio electrónico.

INSTALACIÓN

Para que la instalación y el PrestaShop funcione, tenemos que tener actualizados los repositorios con los siguientes comandos y tener instalado el apache y el mysql en el equipo.

Los comandos para actualizar los repositorios, son los siguientes:

apt-get update apt-get upgrade

El apache lo deberíamos de tener instalado de la primera parte de la wiki.

A continuación instalaremos PHP5 y paquetes extras con los siguientes comandos:

apt-get install php libapache2-mod-php php-cli php-mysql php-mcrypt php-zip php-ldap php-curl php-gd php-odbc php-pear php-xml php-xmlrpc php-mbstring php-snmp php-soap php-intl curl

Una vez terminado el proceso de instalación de todos los paquetes de PHP5, reiniciamos el servicio del apache con el siguiente comando: sudo service apache2 restart

Entraremos en modo MySQL mediante el comando mysql -u root -p

Ahora introduciremos el comando CREATE DATABASE PrestaShop. Después comprobaremos que se haya creado correctamente con el comando SHOW DATABASES.


Ahora crearemos un usuario para la base de datos con el comando CREATE USER adminPrestaShop@localhost IDENTIFIED BY ‘admin’.


Daremos privilegios al usuario recien creado con los siguientes comandos y después mostraremos si los privilegios se han aplicado.

grant all privileges on prestashop * to usrprestashop;
flush privileges;
exit

Lo siguiente que tenemos que hacer es descargarnos la última versión de PrestaShop en el directorio “/tmp”. Para ello, ejecutamos el siguiente comando: cd /tmp && wget https://download.prestashop.com/download/releases/prestashop_1.7.2.4.zip .


Tenemos que instalar el paquete unzip para descomprimir el archivo descargado, para ello con el siguiente comando lo instalaremos: sudo apt-te install unzip -y


Para extraer el contenido del paquete descargado “prestashop_1.7.2.4.zip” en la carpeta “/var/www/html/prestashop” tenemos que crear una carpeta llamada “prestashop” en la carpeta html y después ejecutar el siguiente comando:

creacion carpeta prestashop
mkdir prestashop
descromprimir carpeta
sudo unzip prestashop_1.7.1.4.zip -d /var/www/html/prestashop

Tenemos que aplicarle permisos al directorio de PrestaShop, para ello ejecutamos lo siguiente:

sudo chown -R www-data:www-data /var/www/html/prestashop/

Una vez hechos correctamente los pasos anteriores, para comprobar que todo funciona y poder seguir con la instalación, escribiremos en el navegador la siguiente dirección:

http://10.9.53.105/prestashop/

PROBLEMAS

Nos ha surgido un error que hemos subsanado de la siguiente forma:

Tenemos que habilitar el “módulo de Apache mod_rewrite”, que para habilitarlo, tenemos que seguir los siguientes pasos:

sudo a2enmod rewrite

sudo /etc/init.d/apache2 restart

A continuación editamos el archivo /etc/apache2/sites-enabled/000-default y buscamos la línea AllowOverride None y la cambiamos por AllowOverride All

sudo nano /etc/apache2/sites-enabled/000-default

En nuestro caso, hemos tenido que añadir las siguientes líneas, ya que en nuestro archivo no estaban escritas.

#Include conf-available/serve-cgi-bin.conf
Alias /log/ "/var/log/"

Options Indexes MultiViews FollowSymLinks
AllowOverride All
Order deny,allow
Deny from all
Allow from all
Require all granted


Después de este último paso, reiniciamos el servicio apache con el siguiente comando y seguimos con los siguientes pasos para terminar la instalación.

sudo /etc/init.d/apache2 restart
